# modelo-para-calcular-o-CREDIT-SCORE

Previsão de Credit Score
Este projeto utiliza inteligência artificial para prever a pontuação de crédito (score) de clientes com base em seu perfil socioeconômico.

Objetivo
Desenvolver um modelo de Regressão Linear para estimar o SCORE de crédito a partir de variáveis como idade, escolaridade, renda e bens.

Tecnologias Utilizadas
Linguagem: Python

Bibliotecas: Pandas, NumPy, Scikit-learn, Matplotlib e Seaborn

Estrutura do Código
Carga de Dados: Leitura de base de dados em Excel (dados_credito.xlsx).

Pré-processamento: Tratamento de dados, normalização com StandardScaler e codificação de variáveis categóricas com LabelEncoder.

Treinamento: Divisão entre treino/teste e aplicação do modelo de Regressão Linear.

Avaliação: Análise de performance através de métricas como Erro Quadrático Médio (MSE) e R².
